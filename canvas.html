The giant box above is called a canvas. In HTML, the programming language used to make websites, everything is an element, including plain text, images, buttons, and many more. One of those elements is the canvas, which is just a box of your desired size that you can "paint" on using a programming language like JavaScript. This page was made to contain this canvas for me to learn how to use it.
